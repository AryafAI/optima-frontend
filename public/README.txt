PUT OptimaLogo.png HERE
=======================

The Logo component (src/components/Logo.jsx) loads the file at:

    /OptimaLogo.png   →   public/OptimaLogo.png

Steps:
1. Open Google Drive → Optima folder
2. Download "OptimaLogo.png" to your computer
3. Drag it into THIS public folder (replace this README.txt or leave it,
   doesn't matter — Vite only serves the actual logo file)

If the file is missing, the dashboard automatically falls back to a clean
inline SVG logo, so the app never crashes — but you'll want the real logo
for your demo.

After dropping the file in, refresh the dashboard (Ctrl+Shift+R if it caches).
